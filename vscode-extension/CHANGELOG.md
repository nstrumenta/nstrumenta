# Change Log

All notable changes to the "nstrumenta-vscode" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [2.0.0] - 2025-11-12

### Added
- Model Context Protocol (MCP) integration for AI-assisted development
- Module management commands (list, run, publish)
- Data explorer commands
- Agent monitoring commands
- API key management with secure storage
- Activity bar view for modules
- Status bar integration
- Configuration for MCP server URL

### Changed
- Complete rewrite for modern VS Code extension API
- Updated to TypeScript 4.5+
- Migrated to @modelcontextprotocol/sdk
- Updated activation events to onStartupFinished
- Modernized UI with activity bar integration

## [1.0.3] - Previous Release

### Features
- Basic nstrumenta integration
- Update and watch commands
- Project configuration support
